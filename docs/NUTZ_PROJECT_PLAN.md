# NUTZ — Website Project Plan
## תוכנית בנייה מלאה לאתר תדמיתי | ליאב ברודש

---

## 1. סקירה כללית

**פרויקט:** אתר תדמיתי מולטי-פייג׳ למותג NUTZ — קליסטניקס אונליין
**בעלים:** ליאב ברודש
**מטרת האתר:** לספר את הסיפור של ליאב, לבנות אמון וסמכות, ולהמיר מבקרים למועמדים לתהליך קליסטניקס אונליין
**קהל יעד:** גברים 18+ שרוצים להתחיל או להתקדם בקליסטניקס
**שפה:** עברית (RTL)
**דומיין:** לא רלוונטי עד סיום הבנייה

---

## 2. סטאק טכנולוגי

| טכנולוגיה | תפקיד |
|-----------|--------|
| **Next.js 14+ (App Router)** | פריימוורק ראשי, SSG/SSR ל-SEO |
| **TypeScript** | טיפוסי שפה |
| **Tailwind CSS** | סטיילינג |
| **GSAP + ScrollTrigger** | אנימציות scroll-based ו-page transitions |
| **Netlify** | אחסון ודיפלוי |

### הערות טכניות
- RTL מלא — כל ה-layout בכיוון ימין-לשמאל
- אנימציות GSAP על כל סקשן: reveal on scroll, parallax, stagger
- המסקוט (סנאי NUTZ) כאלמנט אנימטיבי חוזר — אפשר SVG animated
- אוטומציית טופס: אותו מנגנון כמו דפי הנחיתה — נתונים ל-Google Sheets עם תווית "אתר תדמית"
- אין סליקה, אין קורסים, אין אזור אישי — תדמית בלבד

### סגנון כללי:
- **Dark & Powerful** עם אנרגיה של אנימה
- רקע שחור דומיננטי, טקסט בהיר
- אקסנטים בכתום-ערמוני ושוקולד
- זריקות כוח בכחול חשמלי/צהוב לאלמנטים אנימטיביים
- המסקוט (סנאי NUTZ) כאלמנט חוזר עם אנימציות GSAP

### פונטים:
- **כותרות עברית:** Rubik Bold / Black
- **English labels:** Bebas Neue (ל-"THE NUTZ METHOD" וכו׳)
- **Body עברית:** Rubik Regular
- כל הפונטים מ-Google Fonts, תומכים RTL

### לוגו ומסקוט:
- לוגו: אגוז NUTZ על רקע שחור (PNG)
- מסקוט: סנאי שרירי עם גופיית NUTZ — סט שלם של אווטארים

### תמונת Hero:
- **ראשית (דף בית):** IMG_1297 — מבט ישיר למצלמה, שקיעה כתומה-סגולה, ים ברקע
- **משנית (About):** IMG_1294 — פוזת כוח, חיוך למעלה, אותו רקע שקיעה

### OG Image:
- לוגו NUTZ על רקע כהה עם tagline מתחת

### אנימציות GSAP:
- **Page load:** Staggered reveal — לוגו, כותרת, תת-כותרת, CTA
- **Scroll:** כל סקשן עם reveal animation (fade-up / slide-in)
- **Stats counter:** מספרים שעולים באנימציה כשנכנסים ל-viewport
- **Mascot:** הסנאי מופיע באנימציות שונות בכל סקשן
- **Cards:** Hover effects + staggered entrance
- **Parallax:** רקע Hero עם parallax עדין

## 6. Homepage Flow

```
1. HERO          -> Tagline + מספרים + CTA ראשי
2. HOOK          -> משפט פילוסופי שיוצר מתח
3. METHOD        -> טיזר של 3 שלבים (מוביל ל-/method)
4. ABOUT         -> טיזר — תמונה + שם + 3 מספרים (מוביל ל-/about)
5. TESTIMONIALS  -> 2-3 עדויות מקוצרות (מוביל ל-/results)
6. FAQ           -> 3 שאלות שוברות התנגדות בלבד
7. CONTACT       -> טופס סינון
8. FOOTER        -> ניווט + CTA + רשתות חברתיות
```

### עקרונות הסדר:
- Hero מרוויח את ה-Hook
- Hook מרוויח את השיטה
- השיטה מרוויחה את ההכרות עם ליאב
- ליאב מרוויח את ההוכחות (עדויות)
- ההוכחות מרוויחות את שבירת ההתנגדות (FAQ)
- FAQ מרוויח את ההמרה (טופס)

## 8. ארכיטקטורת CTA

| מיקום | סוג CTA | טקסט | כוונת מבקר |
|-------|---------|------|------------|
| Header (fixed) | כפתור ראשי | "להתחיל תהליך" | כוונה גבוהה, כל נקודה |
| Hero | ראשי + משני | "הגשת מועמדות" + "צפה בתוצאות" | רושם ראשון |
| אחרי Method | CTA הנעה | "מוכן להתחיל? מלא את הטופס" | אחרי הוכחה |
| אחרי FAQ | CTA רך | "עדיין יש שאלות? בוא נדבר" | אחרי שבירת התנגדות |
| Footer | כפתור ראשי | "להתחיל תהליך" | fallback ניווטי |
| Floating button | כפתור צף | WhatsApp או "הגשת מועמדות" | fallback קבוע |

CTA ראשי = טופס סינון. לא שיחת טלפון ישירה. יוצר אקסקלוסיביות.

### השיטה (/method)

**HERO:**
- Label: THE NUTZ METHOD
- כותרת: השיטה לשליטה
- תת-כותרת: לא עוד תוכנית גנרית. תהליך מובנה שנבנה סביבך — הגוף שלך, הרמה שלך, המטרות שלך.

**שלבי השיטה (Full):**
- שלב 01 — אבחון אישי: מיפוי יכולות, זיהוי חולשות, הגדרת מטרות ריאליסטיות. נבין בדיוק איפה אתה ולאן אתה רוצה להגיע.
